Detailed Description

Demonstrate drawing histogram as SVG files.

Demonstrate drawing histogram as SVG files. The area under each bin shows the number of cases or value in that bin or class. (For bar charts the height shows the value). Bins can be the same (most common) or different widths. Intervals must not overlap and bins must be adjacent. Various suggestions for choosing good bins have been made, some a common bin width. The example below has different bin widths reflecting very roughly the log of the journey time.
